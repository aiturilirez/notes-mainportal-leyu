# notes-mainportal-leyu

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中的每一个页面均为独立的静态文件，可单独访问或引用。本仓库不针对任何特定网站或域名，仅作为 HTML 内容的集中存放与管理用途。

## 项目简介

本仓库主要服务于以下目的：
- 存储并维护一批独立 HTML 页面
- 提供简单的版本管理与历史记录
- 便于通过 GitHub Pages 或其他静态托管方式发布内容

所有页面均为自包含的 HTML 文件，不依赖外部服务或数据库。

## 目录说明

```
.
├── index.html          # 入口页面（可选）
├── page1.html          # 示例页面
├── page2.html          # 示例页面
└── assets/             # 公共资源文件夹（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

各 HTML 文件可直接通过仓库根路径访问。如有公共资源，建议统一存放于 `assets/` 目录下。

## 页面归档说明

- 每个 HTML 页面均为独立归档，命名应尽量反映页面内容或用途。
- 页面之间不应存在硬编码的跨页面依赖（如相互引用）。
- 如需引用公共资源，请使用相对路径。
- 建议在页面内添加必要的元信息（如标题、作者、日期等），以便识别。

## 维护说明

- 欢迎提交 Pull Request 新增或更新页面。
- 请确保新增的 HTML 文件结构完整、无外部恶意代码。
- 请勿存放包含敏感信息、商业推广或违规内容的页面。
- 本仓库不提供自动化部署或同步服务，如需发布请自行配置。
